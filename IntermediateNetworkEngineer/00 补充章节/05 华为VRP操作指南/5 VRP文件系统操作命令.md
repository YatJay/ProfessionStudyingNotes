# 5 VRP文件系统操作命令

## 常见文件系统操作命令

1. 查看当前目录

```shell
<Huawei>pwd
```
![image-20240309230244982](https://img.yatjay.top/md/image-20240309230244982.png)

2. 显示当前目录下的文件信息

```shell
<Huawei>dir
```
![image-20240309230305744](https://img.yatjay.top/md/image-20240309230305744.png)

Idx：序号

Attr：Attributes，属性

3. 查看文本文件的具体内容

```shell
<Huawei>more
```

4. 修改用户当前界面的工作目录

```shell
<Huawei>cd
```

5. 创建新的目录

```shell
<Huawei>mkdir
```

6. 删除目录

```shell
<Huawei>rmdir
```

7. 复制文件

```shell
<Huawei>copy 
```

8. 移动文件
```shell
<Huawei>move
```

9. 重命名文件

```shell
<Huawei>rename
```

10. 删除文件

```shell
<Huawei>delete
```
