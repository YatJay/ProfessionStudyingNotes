# 类型转换

由于Java是强类型语言所以要进行有些运算的时候的，需要用到类型转换。

```
低-----------------------------------------高
byte,short,char > int > long > float > double
```

运算中，不同类型的数据先转化为同一类型，然后进行运算。

## 注意事项：
1. 不能对布尔值进行转换
2. 不能把对象类型转换为不相干的类型
3. 在把高容量转换到低容量的时候——强制转换
4. 转换的时候可能存在内存溢出问题，或者精度问题
5. 操作比较大的数字的时候注意溢出问题
