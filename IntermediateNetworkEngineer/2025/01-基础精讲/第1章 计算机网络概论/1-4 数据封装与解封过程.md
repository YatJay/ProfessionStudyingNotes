### 各个层传输单元的叫法

| 层次                                       | 封装单元叫法                                    | 标识    |
| :----------------------------------------- | ----------------------------------------------- | ------- |
| 应用层（七层模型的应用层、表示层、会话层） | 协议数据单元PDU(Protocol Data Unit)             | 进程号  |
| 传输层                                     | <font color="red">数据段/数据报</font>(Segment) | 端口号  |
| 网络层                                     | <font color="red">数据包/分组</font>(Packet)    | IP地址  |
| 数据链路层                                 | 数据帧(Frame)                                   | MAC地址 |
| 数据链路层物理层                           | 比特流(Bit)                                     |         |
